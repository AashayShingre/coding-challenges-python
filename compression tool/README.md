## Build your own compression tool

https://codingchallenges.fyi/challenges/challenge-huffman


### To run the app

`python -m app.compress [<json_file1> ...]` 


### To run tests 

`pytest`

If you just want to test the cases from the challenge, run 

`pytest test_compress.py`
